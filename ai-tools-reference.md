##AI Tools Reference

**My personal study guide via git**

_day 1 learning_
- Github is a version control hub
- I can pull a copy of my work locally and make changes there
- Those changes are unsaved
- Then, I can point to the changes I want "used" by using "git add" 
- From there, I can use git commit to save actual changes and narrate what happened
- To get it back into my repository, I use git push to the branch in question

Ultimately, the flow at this point is: 
- git status
- git add
- git diff --staged
- git commit -m 
- git push

Also, I can create notepad like files in .txt or .md using nano. MV is move file. And I can use tab to autocomplete file names.
